# Feature Ideas (Brainstorming)

이 문서는 아직 확정되지 않은 새로운 기능 아이디어나 브레인스토밍 내용을 자유롭게 기록하는 공간입니다.
**아이디어가 구체화되어 개발이 확정되면 `features_list.md`로 이동하고 이 목록에서 제거하십시오.**

## New Ideas

- [ ] **사용자 복습 퀴즈**: 오늘 배운 표현을 간단한 퀴즈로 복습하는 기능.
- [ ] **소셜 공유 이미지 생성**: 멋진 배경에 핵심 표현과 뉘앙스가 적힌 이미지를 생성해 인스타그램/틱톡 공유 유도 (매우 중요).
- [ ] **개인화 학습 경험**: 사용자의 학습 이력을 데이터베이스에 저장하고, 부족한 표현을 AI가 분석해 맞춤형 복습 경로 제공.
